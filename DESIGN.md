# 个人衣橱APP - 设计文档

## 产品定位
一款支持多维度管理与搭配推荐的个人衣橱应用，帮助用户数字化管理衣橱、智能分类整理、自创穿搭组合。

## 设计风格：自然有机风

**气质与意象**：如清晨阳光洒在原木衣柜前，每件衣物都被温柔以待
- 温暖自然的色调，如同衣物本身的质感
- 简洁留白，让衣服成为主角
- 柔和圆角，传递亲和力

**视觉策略**：
- 大地色系主调（米白、浅棕、森林绿点缀）
- 瀑布流展示衣物，突出图片
- 清新自然的图标风格

**配色方案**：
- 主色：#8B7355（温暖棕）
- 背景：#FFFBF5（暖白）
- 卡片：#FFFFFF
- 强调色：#4A7C59（森林绿）
- 文字主色：#3D3D3D（深灰）
- 文字辅色：#8A8A8A（中灰）

**字体排版**：
- 中文：PingFang SC
- 数字：SF Pro Display

**动效与交互**：
- 柔和的弹性动画
- 图片加载渐入效果
- 轻柔的震动反馈

## 页面结构

### 1. 首页（衣橱）
- 顶部搜索栏
- 顶部Tab分类导航：全部/上衣/裤子/外套/裙子/包包/配饰
- 瀑布流展示衣物卡片
- 底部浮动添加按钮

### 2. 拍照上传页
- 相机预览区域
- 拍照按钮
- 相册选择入口
- 抠图结果预览与确认

### 3. 搭配组合页
- 创建新搭配入口
- 我的搭配列表
- 搭配预览卡片（展示组合效果）

### 4. 衣服详情页
- 衣服大图展示
- 分类标签
- 删除/编辑操作

## 核心功能

### 1. 拍照上传与抠图
- 使用expo-camera拍照
- 调用后端AI抠图API处理
- 返回透明背景图片保存

### 2. 电子衣橱存储
- 图片上传到对象存储
- 元数据存储到数据库
- 支持查看、搜索、筛选

### 3. 六维分类管理
- 上衣：T恤、衬衫、卫衣、毛衣等
- 裤子：牛仔裤、休闲裤、短裤等
- 外套：夹克、大衣、风衣、羽绒服等
- 裙子：连衣裙、半裙等
- 包包：背包、手提包、钱包等
- 配饰：帽子、围巾、腰带、首饰等

### 4. 搭配组合
- 选择不同分类的衣服组成套装
- 保存搭配方案
- 预览搭配效果

## 技术架构

### 前端
- Expo 54 + React Native
- expo-router 路由
- expo-camera 拍照
- expo-image-picker 选择图片
- expo-image 图片展示

### 后端
- Express.js
- Multer 文件上传
- AI抠图处理（使用 remove.bg API 或类似服务）

### 数据模型
- Clothing：衣服记录
- Outfit：搭配组合记录
- OutfitItem：搭配中的衣服关联
